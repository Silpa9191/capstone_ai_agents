## Description
This project demonstrates a simple multi-agent system:
- Supervisor agent receives tasks and delegates subtasks
- Worker agent executes tasks or tools
- Includes memory, tool execution, and delegation
- Fully implemented in Python (no external dependencies)

## Usage
1. Save capstone_agents.py and main.py in the same folder
2. Run main.py with Python 3.9+:
   python main.py
3. Observe supervisor, worker, delegation, and tool execution outputs
